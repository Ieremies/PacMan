Para jogar, utilize as teclas WASD do seu teclado!
W move o pacman para cima,
A move o Pacman para a esquerda,
S move o Pacman para baixo e
D move o Pacman para a direita.

Existem 3 mapas, você pode escolher qual deseja digitanto 1, 2 ou 3.
Caso o número digitado seja diferente destes, um mapa aleatório será gerado.

Bom jogo!
